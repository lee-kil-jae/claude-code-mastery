# 개발자 웹 이력서 개발 로드맵

## 프로젝트 개요
개발자를 위한 반응형 웹 이력서를 HTML, CSS, JavaScript, TailwindCSS를 활용하여 개발합니다.

## 기술 스택
- **HTML5**: 시맨틱 마크업 구조
- **CSS3**: 기본 스타일링
- **JavaScript**: 인터랙티브 기능 구현
- **TailwindCSS**: 유틸리티 우선 CSS 프레임워크
